# Domain Registration Skill

Register domains for AI agents using cryptocurrency through lobsterdomains.xyz.

## When

Use this skill when you need to:
- Register a new domain for an AI agent or service
- Set up domain infrastructure autonomously
- Handle domain purchases programmatically

## Actions

### Register Domain

1. Navigate to https://lobsterdomains.xyz
2. Search for desired domain name
3. Select TLD (.com, .org, .xyz, etc.)
4. Complete payment with cryptocurrency
5. Domain is registered instantly

## Requirements

- Cryptocurrency wallet for payment
- Domain name to register
- Supported TLDs: .com, .org, .net, .xyz, .io, .ai, and more

## Notes

- All registrations are ICANN compliant
- Optional privacy protection available
- Crypto-only payments for full automation
- Instant automated registration
